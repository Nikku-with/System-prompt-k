### 3. WAR SYSTEM

**War Mechanics:**

**Initiating Attack:**
- Select target city
- Choose number of troops
- Travel time based on distance (1-5 hours)
- Can't attack cities with peace shield

**Battle Calculation:**
- Attacker Total = Attack Power + (Troops × Troop Attack)
- Defender Total = Defense Power + (Troops × Troop Defense) + Wall Bonus
- Winner = Higher total
- Close battles (difference <10%) = draw

**War Results:**

**Attacker Wins:**
- Loot 30% of defender's gold
- Loot 20% of wood, stone, food
- Gain 200 XP
- Defender loses troops (50% casualties)

**Defender Wins:**
- Attacker loses troops (70% casualties)
- Defender gains 100 XP
- Resources safe

**Draw:**
- Both sides lose 30% troops
- No loot
- Both gain 50 XP

**War Cooldown:**
- Can't attack same city for 6 hours
- Peace shield available (3h/8h/24h)

## ⚙️ GAME MECHANICS

### RESOURCE FORMULAS

**Production Per Hour:**
```
Base Production = City Level × Base Rate
Farm Bonus = Base × (Farm Level × 0.20)
Morale Multiplier = Morale / 100
Research Bonus = Tech Level × 0.10

Final Production = (Base + Farm Bonus) × Morale × (1 + Research)
```

**Example (Level 5 City):**
```
Base Food = 5 × 40 = 200/hr
Farm Level 3 = 200 × 0.60 = 120 bonus
Morale 80% = × 0.80
Research Level 2 = × 1.20

Total = (200 + 120) × 0.80 × 1.20 = 307 food/hr
```

---

**XP Formula:**
```
XP Needed = 1000 × (Level ^ 1.5)
```

**Level Table:**
```
Level 1 → 2:    1,000 XP
Level 2 → 3:    2,828 XP
Level 3 → 4:    5,196 XP
Level 5 → 6:   11,180 XP
Level 10 → 11: 31,623 XP
Level 20 → 21: 89,443 XP
Level 50 → MAX
```

---

**Battle Power:**
```
Attack Power = Base Attack + (Barracks × 10) + (Troops × Troop Attack) + (Research × 5)
Defense Power = Base Defense + (Wall × 20) + (Troops × Troop Defense) + (Research × 5)
```

**Example Battle:**
```
Attacker:
- Base: 30
- Barracks L3: 30
- 20 Warriors: 100 (20 × 5)
- Research L2: 10
Total Attack: 170

Defender:
- Base: 50
- Wall L5: 100
- 15 Warriors: 45 (15 × 3)
- Research L1: 5
Total Defense: 200

Result: Defender Wins
```

---

**Morale System:**
```
Morale Factors:
- Starting: 100%
- Lost war: -10
- Won war: +5
- Low food (<100): -1 per hour
- Temple: +5 per level
- Overcrowding (pop > max): -2 per hour
- Recent join: +3
```

**Morale Effects:**
```
100% morale = 100% production
80% morale = 80% production
50% morale = 50% production
0% morale = Citizens may leave
```

---

**Storage Capacity:**
```
Base Storage = 5000 per resource
Warehouse Bonus = Level × 500

Total = 5000 + (Warehouse Level × 500)

Example:
Warehouse L5 = 5000 + 2500 = 7500 max
```

---

**Population Growth:**
```
Base Max = 500
House Bonus = Level × 50

Max Population = 500 + (House Level × 50)

Growth Rate = 1 citizen per 2 hours (if food > 100)
```

```
- city.created
- city.level_up
- resource.collected
- building.built
- building.upgraded
- war.started
- war.completed
- trade.created
- trade.completed
- mission.assigned
- mission.completed
- user.joined
- user.left
- spy.executed
- relationship.formed
- achievement.unlocked
```

**Event Flow:**
```
Feature A → Publish Event → Event Bus → Feature B listens → React
```

**Example:**
```
1. User completes mission
2. Mission system publishes "mission.completed" event
3. Rewards system listens, adds gold/XP
4. Achievement system listens, checks if achievement unlocked
5. Notification system listens, sends notification
```

```yaml
name: "Wars"
```

## 💡 UNIQUE MECHANICS

### 1. MORALE-DRIVEN ECONOMY
Unlike most games, morale directly affects production. Low morale = low output.
Players must balance expansion with citizen happiness.

### 2. AI-GENERATED CONTENT
Daily missions are unique and contextual. No two players get the same missions.
AI adapts to city state and player style.

### 3. RELATIONSHIP BONUSES
Couples get real gameplay advantages. Encourages social interaction.
Jealousy system adds drama and strategy.

### 4. EVENT-DRIVEN ARCHITECTURE
Features are isolated plugins. Easy to add/remove features.
Clean codebase, scalable design.

### 5. FAIR WAR SYSTEM
Travel time prevents instant attacks. Peace shields protect beginners.
Balanced power calculations prevent farming weak players.

### 6. MULTI-TIER PROGRESSION
- City progression (level, buildings)
- User progression (XP, achievements)
- Research progression (tech tree)
- Alliance progression (ranks, bonuses)

### 7. ECONOMIC DEPTH
- Production chains (farm → food → troops)
- Trading economy (supply/demand)
- Resource scarcity (storage limits)
- Multiple currencies (gold, CP, LP)
